# Desafio Git: Clonado y Remotos

---

## ¿Como clonar el repositorio?
### Debemos crear el repositorio en *Git*, posteriormente; accedemos a la pestaña de *Git* y seleccionamos "Clone" <br/> Copiamos la URL del repositorio y seleccionamos la carpeta donde queremos que este ubicado.
![Clonar1](/imagen/Clonar1.png)

---

## ¿Como Configurar Usuario?
### Al entrar en configuracion y commit podemos configurar el usuario.
![Clonar2](/imagen/Clonar2.png)

---

## ¿Como ver el nombre y ubicacion del repositorio actual?
### En la pestaña "main" vemos donde se ha guardado el repositorio y cuales otras ramas tenemos.
![Clonar3](/imagen/Clonar3.png)

---

## ¿Como se suben los cambios al repositorio remoto?
### En la seccion izquierda, seleccionamos el documento que queremos subir <br/> Para luego poder hacer *"Commit"* y *"Push"* <br/> Asi se sube el codigo al repositorio remoto.
![Clonar4](/imagen/Clonar4.png)
